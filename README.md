# CCAF Practice

Interactive practice app for the **Claude Certified Architect – Foundations** exam.

A single self-contained HTML file (`index.html`) — open it in any browser, no internet or API key needed.

## What's inside

- **326 scenario-style questions** across all 5 official domains, weighted to match the exam blueprint
- All **12 official sample questions** from the Anthropic exam guide
- Questions authored from the official task statements plus community resources ([certsafari.com](https://www.certsafari.com/anthropic/claude-certified-architect), [claudecertifications.com](https://claudecertifications.com/), Balaji Ashok Kumar's [Medium guide](https://dynamicbalaji.medium.com/claude-certified-architect-foundations-certification-preparation-guide-c70546b51f51), and Daron Yondem's [exam prep guide](https://github.com/daronyondem/claude-architect-exam-guide))

## Modes

- **Random shuffle** — weighted by exam blueprint, endless rotation
- **By domain** — drill a single domain
- **Timed exam** — simulates the real 65-question / 90-minute exam with pass/fail at scaled score 720
- **CertSafari exam** — separate full exam mode using only CertSafari questions
- **Quick 10** — fast spot-check

## Features

- Instant feedback after each submit with the correct answer and a detailed explanation
- Score and per-domain breakdown at the end
- Answer order shuffled each session so you cannot memorize positions
